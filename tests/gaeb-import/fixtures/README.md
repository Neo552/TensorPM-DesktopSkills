# GAEB test fixtures — provenance

Official, publicly downloadable GAEB interoperability/certification sample files.
They carry no explicit license but are published by GAEB e.V. / BVBS expressly for
interop and certification testing and are already redistributed in MIT-licensed
open-source repos. Source URLs below; do not modify the files.
